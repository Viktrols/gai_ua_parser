Задача: Напишите функцию, которая берет номер автомобиля как параметр и выдает данные по этому автомобилю, полученные через API.
1.	Перейдите по ссылке https://baza-gai.com.ua/api и получите бесплатный API для подключения к базе автомобилей.
Здесь же вы можете ознакомится с тем, как работать с API.
2.	Получите данные по машинам со следующими номерными знаками: "CE1789BA", "CE1786BA", "CB8136AX", "AE8181EX".
Важно: если вы используете Python для решения задания, в функции requests.get тут помимо url требуется еще второй параметр: headers=headers.
3.	В результате вы должны получить json файл с данными о регистрации машин.
Ваша задача распарсить его и сделать пригодным для анализа. Стоит отметить, что в этом json'е есть вложенные словари,
которые также нужно спарсить. Убедитесь, что 1 номерной знак = 1 ряд, не содержащий словарей.
4.	Избавьтесь от рядов с отсутствующими номерами машин, если такие есть.
